Project Overview

What this analysis sets out to answer

Cell Painting is a high-content imaging assay that captures the morphological response of cells to chemical perturbations. This project asks: can we discover biologically meaningful compound groupings in a fully unsupervised way — using only the Mito channel — and do those groupings agree with an independent supervised classifier?

We process every well image through a self-supervised DINOv3 Vision Transformer, aggregate features at the compound level, apply Harmony batch correction, and cluster the resulting embedding with the Leiden algorithm. The answer is a clear yes: 16 out of 20 of a classmate's top supervised hits land in the same unsupervised cluster, and the best cluster shows 24× enrichment for high-confidence hits over random expectation.

Data & Preprocessing

JUMP Cell Painting — Mito channel

The dataset comprises 407 plates from the JUMP Cell Painting consortium. Each well contains a 16-bit TIFF image of the mitochondrial (Mito) channel. Images are read directly as float32 — no lossy 8-bit conversion — and normalized using percentile clipping (1st–99th) to handle intensity variation across plates.

Virtual Tiling Strategy

Rather than resizing images (which loses local morphological detail), we apply virtual tiling: each image is split into a 3×4 grid of 12 tiles. Each tile is independently resized to 224×224 px for the DINOv3 backbone. This preserves cellular-level resolution while fitting the model's expected input.

Tile & Resize Comparison

Original image vs. virtual tiles at 224×224 px. Tiling preserves cellular morphology that naive downscaling would destroy.

Feature Extraction

Self-supervised vision transformer features

We use DINOv3 ViT-B/16 (facebook/dinov3-vitb16-pretrain-lvd1689m), a self-supervised Vision Transformer pre-trained on 1.7 billion images. Each tile produces a 768-dimensional feature vector from the [CLS] token. Tile features are mean-pooled to get image-level features, then aggregated to well-level (mean) and finally to compound-level (median across replicate wells).

Why DINOv3? Self-supervised pre-training learns general visual features without task-specific labels. DINOv3 captures both texture and structural patterns, making it well-suited for morphological profiling where the phenotype is not known a priori.
Feature Selection

After aggregation, features with variance < 0.01 are removed, followed by dropping one of each pair with Pearson r > 0.95. Remaining features are L2-normalized.

Batch Correction

Harmony integration across 407 plates

Plate-level batch effects are a known confound in Cell Painting. We apply Harmony at the well level, using plate IDs as the batch variable. The algorithm iteratively clusters wells and computes per-plate correction factors using soft k-means, converging when the objective stabilizes.

Batch Correction Diagnostics

iLISI = 18.76 indicating moderate plate structure. Harmony reduces plate-driven clustering while preserving biological signal.

With vs. Without Harmony

Fair comparison at identical Leiden resolution. Without Harmony, fold enrichment is higher (23×) but batch effects confound the signal.

Trade-off: Harmony flattens some genuine biological variation along with batch effects. We accept this trade-off because plate-level confounds would otherwise dominate the clustering.

Clustering Results

Leiden community detection on the UMAP embedding

We construct a k-nearest-neighbor graph (k=30, cosine distance) on the batch-corrected features and apply the Leiden algorithm at resolution 0.5, yielding 22 clusters. UMAP is used for 2D visualization (n_neighbors=30, min_dist=0.1).

Hit Discovery & Validation

Unsupervised clusters capture supervised hits

An independent supervised classifier identified 96 high-confidence hits (confidence ≥ 0.85). We overlay these on our unsupervised UMAP to test whether biologically active compounds cluster together without any label information.

Supervised Hits on Unsupervised UMAP

96 supervised hits (red) overlaid on the unsupervised UMAP. Hits concentrate in specific clusters rather than scattering uniformly — strong evidence that unsupervised features capture biologically meaningful variation.

Key finding: Cluster 16 contains 50 out of 96 supervised hits (52%), despite holding only ~2% of all compounds — a 24× enrichment over random expectation. Among the classmate's top 20 hits, 16 out of 20 (80%) land in cluster 5.

Model Comparison

DINOv3 vs. EfficientNet vs. Hybrid features

To validate our feature extractor choice, we compare three approaches: DINOv3 ViT-B/16 (self-supervised), EfficientNet-B0 (supervised ImageNet), and a hybrid concatenation of both.

UMAP Comparison Grid

UMAP embeddings for all three methods. DINOv3 produces the most distinct cluster boundaries, while EfficientNet yields slightly more granular groupings.

Quality Metrics Comparison

Quantitative comparison across clustering quality metrics. All methods show consistent phenotypic groupings, supporting the robustness of the approach.

Conclusion: All three feature extractors capture similar biological structure. DINOv3 was selected as the primary method for its stronger cluster separation and richer representation of morphological variation.

Interpretability — Grad-CAM

What the model looks at to distinguish phenotypes

We train a lightweight classifier on top of the extracted features and use Grad-CAM to visualize which image regions drive cluster assignment. This provides biological interpretability: do the model's decisions correspond to recognizable cellular structures?

Grad-CAM Summary

Grad-CAM attention maps for representative wells from multiple clusters. The model consistently attends to mitochondrial morphology — fragmented, elongated, or punctate patterns — confirming biological relevance.

Biological relevance: Grad-CAM reveals that clusters correspond to distinct mitochondrial morphologies: fragmented networks (stress response), elongated filaments (healthy state), and punctate foci (potential apoptotic signals).
